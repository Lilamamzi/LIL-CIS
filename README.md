# Lil CIS — راهنمای هاست رایگان

این نسخه کاملاً سمت مرورگر (client-side) هست — هیچ سروری، هیچ Agent AI‌ای، و هیچ محدودیت اعتباری نداره.
دیتا داخل خود مرورگر گوشی/کامپیوترتون (localStorage) ذخیره می‌شه.

## گزینه ۱ — GitHub Pages (پیشنهادی، کاملاً رایگان و دائمی)
1. یه اکانت گیت‌هاب بسازید (اگه ندارید): github.com
2. یه ریپوی جدید بسازید (مثلاً `lil-cis`)
3. همه فایل‌های این پوشه (index.html, style.css, app.js, ui.js, data.js, charts.js, ocr.js, manifest.json, sw.js, پوشه icons) رو آپلود کنید (از دکمه "Add file → Upload files")
4. برید Settings → Pages → Source رو بذارید رو "main" branch، پوشه "/ (root)"
5. بعد چند دقیقه، آدرسی شبیه `https://username.github.io/lil-cis/` فعال می‌شه.
6. از گوشی با Chrome برید همون آدرس، "Add to Home Screen" بزنید.

## گزینه ۲ — Cloudflare Pages (رایگان، کمی سریع‌تر برای ست‌آپ)
1. cloudflare.com → Pages → Create a project → Direct Upload
2. کل پوشه رو drag & drop کنید
3. یه آدرس `.pages.dev` می‌گیرید — همیشگی و رایگان

## نکات مهم
- **بک‌آپ:** چون دیتا فقط رو مرورگر خودتونه، از تب «تنظیمات» به‌طور مرتب «دانلود پشتیبان» بزنید و فایل JSON رو یه‌جای امن نگه دارید (مثلاً همینجا برای من بفرستید).
- **دو گوشی/دو حساب:** اگه از دو گوشی مختلف استفاده می‌کنید، دیتا خودکار sync نمی‌شه. باید از همون گوشی که همیشه استفاده می‌کنید کار کنید، یا با Export/Import بین گوشی‌ها منتقل کنید.
- **آپدیت بعدی:** هر وقت خواستم فایل‌ها رو آپدیت کنم (مثلاً موتور جدید)، فقط کافیه همون فایل‌های تغییریافته رو دوباره تو گیت‌هاب/Cloudflare آپلود کنید — نیازی به ساخت پروژه جدید نیست.
